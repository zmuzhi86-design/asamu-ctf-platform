# asamu 0.3.1 正式修复版

## 主要更新

- 本地部署默认启动动态容器 Worker，并验证数据库心跳与宿主机镜像发现。
- 修复 Worker 首次维护失败或 Docker 镜像列表阻塞时无法在线注册的问题。
- 首页背景改为淡化透明纹理，白色遮罩按正确层级覆盖素材。
- 动态容器镜像输入框初始为空，不显示任何默认文字。
- UUID 动态 Flag 使用 `flag{xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}` 格式。
- 动态容器宿主端口池扩大为 `20000-30000`。
- TCP/UDP 环境显示纯 `IP:端口`，HTTP/HTTPS 保留可点击 URL。
- 首页和题库默认启用 Web、Misc、Reverse、Mobile、Pwn、IoT、Crypto 七个方向，其余方向默认归档。
- 全站替换为 0.3 正式版浅色像素安全主题背景。
- 队长可以上传 PNG、JPEG、WebP 战队头像，限制 5 MB 和 4096×4096。
- 用户资料支持头像选择，题目管理支持标准随机或 UUID 动态 Flag。

## 升级说明

升级脚本会自动执行数据库迁移，并将现有运行时端口配置更新为 `20000-30000`。请同时在服务器防火墙和云安全组中放行 TCP/UDP `20000:30000`。

升级已有安装时不要使用 `--fresh`，否则会删除现有业务数据。
