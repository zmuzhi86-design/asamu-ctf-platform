# asamu v3 视觉检查记录

检查日期：2026-07-11

## 实际检查页面

- 首页、题库/探索方向、训练路线
- 战队列表、战队详情、比赛中心
- 题目详情、动态靶场实验舱
- 排行榜、个人主页
- 管理端素材中心、页面槽位、背景中心

## 检查宽度

- 1440 × 900
- 1280 × 800
- 768 × 900
- 390 × 844

上述目标页面均通过 DOM 尺寸检查：没有横向滚动，已加载图片没有破图。

## 已修复问题

- 清除原图棋盘格假透明背景，导出带真实 Alpha 的 WebP。
- 首页、比赛、战队移除重复机器人、奖杯和旧主视觉叠加。
- 默认素材统一使用 `contain`，避免人物、徽章与场景被错误裁切。
- 战队详情 Banner 单独使用 `cover` 与视觉焦点，避免主体过小。
- 标签与机器人提示卡增加自适应对齐，避免在 Grid/Flex 中被拉成长条或整列高度。
- 背景增加低透明遮罩，避免影响标题和正文可读性。
- 动态环境运行态只保留“重新启动 / 关闭环境 / 复制地址”，重置移入次级菜单。
- 管理端透明预览同时显示浅色与深色底，便于发现白边、灰边与残片。

## 人工确认

- 生产对象存储/CDN 域名、素材审核权限和病毒扫描策略。
