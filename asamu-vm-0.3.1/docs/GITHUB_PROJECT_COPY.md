## 4. 第一次公开发布文案（推荐）

今天正式发布我的第一个 GitHub 项目：**asamu** 🎉

这是我从零开始持续搭建的一套 CTF 教学与竞赛平台。项目采用 React + TypeScript、Go、PostgreSQL、Redis 和 Docker Compose，已经实现题库、比赛、战队、排行榜、Writeup、学习中心、管理后台，以及每名用户独立的 Docker 动态靶场和动态 Flag。

这个项目对我来说不只是一个网页，也是我学习网络安全、CTF、前后端开发、数据库、Docker 和 Linux 运维过程的一次完整实践。

当前版本是 **v0.3.1**。它还有很多可以继续改进的地方，但我决定先把它公开出来，记录自己的成长，也希望获得真实的建议和反馈。

欢迎大家查看、Star、提交 Issue，也欢迎指出部署、功能和安全设计上的问题。感谢每一位愿意认真体验和提出建议的人。🙏

## 5. 简短发布文案

我的第一个 GitHub 项目 **asamu** 正式发布 🎉

一套基于 React、Go、PostgreSQL、Redis 与 Docker Compose 的 CTF 教学和竞赛平台，支持题库、战队、比赛、Writeup、排行榜、动态 Docker 靶场与动态 Flag。

这是我学习网络安全和全栈开发过程中的阶段性成果。当前版本 v0.3.1，欢迎 Star、Issue 和建议。

## 7. Release 简介

```text
这是 asamu 的首个 GitHub 公开版本。

v0.3.1 已包含完整的 CTF 平台基础能力：题库、学习中心、比赛、战队、排行榜、Writeup、管理后台、素材系统，以及基于 Docker Worker 的用户独立动态靶场和动态 Flag。

本版本重点修复了 Worker 注册、端口分配、运行时状态恢复和页面背景显示问题，并将默认动态端口池扩大至 20000-30000。
```
