# asamu Platform 素材使用清单

## 处理原则

- 原始素材完整保存在 `apps/web/public/assets/raw/`，没有覆盖或改写。
- `scripts/process_assets.py` 将素材合集裁切为 102 个独立 PNG，并对边缘棋盘格执行透明化处理。
- 处理后素材统一放在 `apps/web/public/assets/processed/`。
- 页面只通过 `apps/web/src/data/assetManifest.ts` 的语义名称引用，不在 JSX 中散落硬编码路径。
- 按钮、输入框、卡片、表格、分页和 Toast 使用 React + Tailwind 实现，不把 UI 截图当组件使用。

## 分类目录

- `characters/`：机器人、学生、引导角色。
- `category-icons/`：Web、Pwn、Reverse、Crypto、Misc、Forensics、IoT、Mobile、Cloud、AI Security。
- `category-scenes/`：各方向专题实验场景。
- `banners/`：学院、比赛舞台、训练地图、实验室背景。
- `environment/`：未启动、启动中、运行中、即将过期、已过期、重置、失败、维护。
- `flag-feedback/`：正确、错误、一血、重复、格式错误、处理中、完成。
- `competitions/`：领奖台、奖杯舞台、倒计时、First Blood。
- `teams/`：旗帜、徽章、队长、头像框、公告板、基地、荣誉。
- `achievements/`：奖杯、奖牌、等级、经验、徽章、宝箱。
- `empty-states/`：空数据、成功、失败、维护、搜索为空、网络异常、锁定。

## 暂未使用素材

- 夜间模式专用素材合集中的部分小装饰未单独裁切：当前夜间主题通过 Token 切换，避免为了装饰降低正文可读性。
- UI 组件级素材重制版仅作为视觉参考，没有直接当图片使用：组件必须保持真实交互与响应式能力。
- Blockchain、OSINT、ICS 场景已完成裁切但暂未进入主导航：第一版题库方向按需求中的 10 个核心方向展示。
- 部分重复奖章、边框角饰和小图标未使用：当前页面已达到足够的像素趣味密度，继续加入会造成视觉噪声。

## 复现处理

在项目根目录运行：

```powershell
python scripts/process_assets.py
```

脚本需要 Pillow；它不会修改 `raw` 中的原始文件。
