# asamu 前端素材目录

原始压缩包保留在项目根目录，运行时不直接引用素材总表图片。

- `source/`：原始文件的可追溯副本，仅供处理使用。
- `sprites/characters/`：机器人、人物和助手。
- `sprites/categories/`：CTF 方向图标。
- `sprites/status/`：动态环境与 Flag 反馈。
- `sprites/achievements/`：等级、奖杯、战队徽章。
- `sprites/decorations/`：角标、分隔线与小装饰。
- `banners/`：首页、比赛和训练路线场景。
- `ui-reference/`：组件视觉参考，不作为整张 UI 图片上线。

素材需要先裁切并处理为真实透明 PNG/WebP；上传包内的棋盘格不是透明通道。
